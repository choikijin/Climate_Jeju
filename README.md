The temperature and relative humidity of Jeju Island from 2017 to 2021 is in 'Env_Jeju.csv'. These data are downloaded from the Korea Meteorological Administration(KMA) "Open MET Data Portal" (https://data.kma.go.kr/cmmn/main.do).

The SFTS incidence per 100,000 population was accessed from publicly available sources at the Korea Disease Control and Prevention Agency "Infectious Disease Data Portal" (https://dportal.kdca.go.kr/pot/index.do).  

The projection by SSP scenarios can be provided upon request from the Korea Meteorological Administration(KMA) Climate Information Portal (http://www.climate.go.kr/home/).
